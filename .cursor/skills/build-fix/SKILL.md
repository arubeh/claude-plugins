# Build Fix スキル

ビルドエラーを段階的に修正する。

## 使い方

```
/build-fix
```

## 実行内容

1. **ビルド実行**: 設定ファイル（`package.json`, `Makefile`, `Cargo.toml`, `build.gradle` 等）からビルドコマンドを判定して実行
2. **エラー解析**: 出力をファイル別にグループ化、重要度順ソート
3. **修正ループ**: 各エラーについて
   - エラーコンテキストを表示（前後数行）
   - 問題を説明し修正を適用
   - ビルドを再実行して解決を確認
4. **停止条件**: 修正が新エラーを導入 / 同一エラーが3回持続 / ユーザーが停止

**一度に1つのエラーのみ修正する。**

## 連携

- `/issue-flow` Phase 3 でビルドエラー時に使用
- `/tdd` 実装中のビルドエラー修正
- `/code-review` 前にビルド成功を確認
