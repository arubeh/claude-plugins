# Build Fix スキル

ビルトインの **build-error-resolver** エージェントを呼び出し、ビルドエラーを段階的に修正します。

## 使い方

```
/build-fix
```

## 実行内容

1. ビルド実行: プロジェクトのビルドコマンドを自動検出して実行
   - 設定ファイル（`package.json`, `Makefile`, `Cargo.toml`, `build.gradle` 等）からビルドコマンドを判定
2. エラー出力を解析（ファイル別にグループ化、重要度順ソート）
3. 各エラーについて:
   - エラーコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を適用
   - ビルドを再実行
   - 解決を確認
4. 停止条件:
   - 修正が新しいエラーを導入
   - 同じエラーが3回試行後も持続
   - ユーザーが一時停止を要求

安全のため、一度に1つのエラーを修正。

## 使用エージェント

| エージェント | 種類 | モデル |
|-------------|------|--------|
| build-error-resolver | ビルトイン | inherit |
